# GTJA Alpha191 — 顶档因子在「歌者」universe 上的实测

> 数据：本地 cache（1489 只股票 × 8691 个交易日）
> 跑出时间：2026-06-26 10:39
> 成本：双边 0.127% / 笔（与 `factor_portfolio_backtest` 一致）
> 因子集：12 个（GTJA 191 Top 9 IR + 3 对照）

## 样本说明 / Caveats

本次跑的是 **TDX 满历史 cache**：日期并集 8691 天（1991→2026，单只票随上市时间不等），有效横截面从 2000 年前后开始（早期上市数 < MIN_STOCKS 阈值会被跳过）。**这是多周期验证**，覆盖了
2008 金融危机、2015 杠杆牛/股灾、2018 熊市、2020 疫情、2024-25 反弹。

**结论的强度分两档**：
- **结构性结论（强）**：相关性簇划分在 266 天窗口与满历史窗口下**几乎一致**（见下文对照），说明因子的语义同源关系与市场环境无关，可直接用于去冗余。
- **回测 α（中）**：含成本回测（双边 0.127%）多周期下**1/12 跨过成本门槛**。绝对水平受「全市场等权基准」口径影响
  （基准含小市值、长期年化偏高），α 为负更多说明「裸因子单腿打不过等权小盘」，
  而非因子无信息（IC 多周期显著）。

### 几个非显然的读出

1. **alpha32 / alpha16 / alpha120 / alpha2 是同源四簇**（|ρ| 0.63–0.88）——
   全部基于「日内 K 线位置 (C-L) vs (H-C)」或其差分/排名变体。原研报把它们
   按 IR 排进 Top 9 时**没有做 VIF/聚类**——只要四个之一进策略池就够了。
2. **alpha74 / alpha70 是「波动 / 量能」对**（ρ=+0.82）——分子都是 close 波动或
   涨幅，分母都是 mean(VOL)。结构同源，也只该留一个。
3. **alpha1 / alpha99 ρ=-0.40 是反向兄弟**——alpha1 看「量价反转」，
   alpha99 看「量价同步」，本就互为镜像，组合时一正一反等于抵消。
4. **真正正交（且有效）的少数派**：alpha62 / alpha83 / alpha90 / alpha176 与多数
   因子 |ρ|≤0.2，这四个才是「191 Top 9 里**独立**贡献信息的部分」。
   ↑ 这是原文章漏掉的最关键判断——Top 9 不是 9 个独立信号，是 ~4-5 簇。

在我们 0.127% 双边成本下，IC=0.03 的因子未必能净赚（详见 [[factor-ic-findings]]）。

## 结论

- **跨过 0.127% 成本门槛**（H5 α>0）：1/12 个因子
  - `alpha1` α=+2.6%
- **被成本吃掉**：11/12 个因子
  - `alpha83`, `alpha99`, `alpha62`, `alpha90`, `alpha32`, `alpha16`, `alpha176`, `alpha74`, `alpha70`, `alpha2`, `alpha120`

**给「歌者」的落地建议**：

1. **相关性去冗余立即可用**：alpha32/16/120/2 四簇取一、alpha74/70 二取一、
   alpha1/99 互为镜像——这是原研报漏掉的步骤，且在两个窗口下结论一致。
2. **多周期 IC 显著但 net α 仍负** → 裸因子不能直接当 top-q 多头策略上线。
   正确用法是把 IC 显著的独立因子（alpha62 / alpha83 / alpha90 / alpha1）
   做**横截面排序打分 + 多因子线性合成**，再叠加择时/成本控制，而非单因子满额换手。
3. **成本敏感性**：当前已降到 0.127% 双边（2026-06-26 实盘费率调研后，佣金万0.854+过户规费万1+印花税+0.02%滑点）。即便如此多周期回测的通过数见上——
   与 [[factor-ic-findings]] 互证：A 股裸量价因子能否打过等权小盘基准对成本高度敏感。
4. 与 `docs/backtest_report.md` 的策略级回测互证：这里是「因子能不能成为策略」，
   那边是「策略能不能上线」，两套验证不冲突。
