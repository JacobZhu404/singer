## 与 800 抽样快验相比的关键校正

### 1. 抽样把稀疏策略排名带偏了 —— 全量翻盘

抽样里 tail_market(62 笔)、right_side(57 笔)笔数太少，α 被低估排在 4/7 名。
全量下笔数升到 302/257，二者实为 **30d α 第 1/第 2**。rps_breakout 抽样 +9.42 是
高估，全量回到 +6.11（与 tail/right 挤在 6% 一档）。

→ **教训复用**：稀疏策略的抽样 α 不可信，正是 `MEMORY.md` 里
`feedback-stale-backtest-decisions` 强调"不可逆决策必须等全量"的实证。

### 2. chanlun_strict 中游确认 —— 弃用案彻底作废

全量 30d α **+4.85（5/16 中上游）**、10d **+0.72**、win **52.3%（全场最高）**。
旧 0.31% 成本口径下它是唯一 10d 转负、30d 垫底；新成本 0.127% 把这个 500+ 笔的
高换手策略从水下托回中游。**此前"降级为弃用候选"的决定正式撤销**。
（score 高分段反向 bug 仍可选修，那是"优化"不是"弃用"。）

### 3. 新的最弱坐实为 bollinger_lower_bounce

下轨反弹（均值回归）全量 30d α **+1.39（16/16 垫底）**，5d/10d 仅 +0.06/+0.14——
均值回归的小幅反弹基本被 0.127% 双边换手吞光。它在 G2 反转簇里被 reversal
（30d +2.63、10d +1.14）全面压制。**它是唯一确凿的优化/弃用对象**（详见下方）。

### 4. 中持有期 10d α 转负的只剩两个

limit_up_gene（-0.16）、golden_cross（-0.09）。抽样里担心的 macd_bull/td_sequential
10d α 在全量都转正（+0.73 / +0.36），警报解除。

## 与 IC 因子工具的关系

本报告是**策略级**回测，每个策略是"信号过滤器 + 评分"的复合体。
**因子级**的 IC/IR 验证见 `tools/ic_validation.py`、`tools/factor_portfolio_backtest.py`
与 `tools/gtja_factor_validation.py`。

**两者结合的方法论**：因子 IC 用来筛"该不该写成策略"，策略回测用来定"上不上线"。
共同教训：A 股高换手信号的 α 高度依赖成本口径与样本完整度——0.18pp 费率差能让
chanlun 从垫底到中游，抽样 vs 全量能让 tail_market 从中游到第一。
