# 歌者智能选股系统 - 问财回测策略查询语句
# 复制每行到问财平台 (https://backtest.10jqka.com.cn) 进行回测
# 建议回测周期：2024-01-01 至 2025-12-31
# 建议持股周期：5,10,20 天

================================================================================
策略1: MACD多头排列 (macd_bull)
================================================================================
MACD金叉; DIF大于0; DEA大于0; 5日均线大于10日均线; 10日均线大于20日均线; 20日均线大于60日均线; 收盘价大于5日均线

================================================================================
策略2: 强势股选股 (strong_stock) - 优化后
================================================================================
量比大于1.5; 收阳线; 近10日涨时成交量除以跌时成交量大于2; 连续5日收阳线且每日涨跌幅绝对值小于等于3%且5日累计涨幅小于12%; MACD零轴以上

================================================================================
策略3: 神奇九转 (td_sequential)
================================================================================
TD序列等于9; 成交量大于5日均量1.2倍

================================================================================
策略4: 右侧交易 (right_side) - 优化后
================================================================================
突破20日最高价; 量比大于1.5; 突破60日最高价; 股价大于60日均线; RSI大于等于50且小于等于70; MACD零轴以上; 5日涨幅大于等于3%且小于等于25%

================================================================================
策略5: RSI超卖 (rsi_oversold) - 优化后
================================================================================
RSI小于30; 或RSI前一日小于30且当日RSI大于等于30小于40; 收盘价小于20日均线

================================================================================
策略6: 布林带反弹 (bollinger_bands) - 优化后
================================================================================
股价触及布林带下轨3%以内; 或前一日触及布林带下轨2%以内且当日收阳线; 股价大于20日均线5%

================================================================================
策略7: 量价突破 (volume_breakout)
================================================================================
量比大于等于2; 或量比大于等于1.5; 收盘价突破30日最高价

================================================================================
策略8: 涨停基因 (limit_up_gene)
================================================================================
近20日有涨停板; 近5日涨时成交量除以跌时成交量大于1.5; 量比大于1.2; 股价大于20日均线

================================================================================
策略9: 缠论中枢 (chanlun)
================================================================================
底分型; MACD底背离; 或价格触及中枢下沿; 或价格突破中枢上沿; SKDJ小于25

================================================================================
策略10: 均线金叉 (golden_cross)
================================================================================
5日均线上穿10日均线; 5日均线大于10日均线大于20日均线; 收盘价大于5日均线; RSI大于等于50小于等于65; MACD零轴以上

================================================================================
组合策略建议
================================================================================
# 高胜率组合：MACD多头 + 右侧交易
MACD金叉; DIF大于0; 5日均线大于10日均线; 突破20日最高价; 量比大于1.5

# 超跌反弹组合：RSI超卖 + 布林带
RSI小于30; 股价触及布林带下轨3%以内; 成交量萎缩

# 强势股启动：强势股 + 右侧交易
量比大于1.5; 近10日涨时成交量除以跌时成交量大于2; 突破20日最高价; MACD零轴以上
